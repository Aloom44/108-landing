Regular price for customers partner_price # Bulk price for partners (discount) stock, sku, image, category ``` ### Order ⭐ (CENTRAL) ```python source_type = 'direct' | 'affiliate' | 'partner' status = 'pending' → 'confirmed' → 'shipped' → 'delivered' affiliate (FK, nullable) # Who referred this order partner (FK, nullable) # Which partner created this order commission_amount = 0 until delivered is_commission_paid = boolean ``` ### Affiliate ```python affiliate_code (unique) # Use in ?ref=code balance # Commission earnings commission_percentage # Per order rate is_active # Can earn commissions? ``` ### Partner ```python company_name bulk_discount_percentage # Discount on bulk orders minimum_order_quantity # Min items for bulk pricing is_verified # Can place bulk orders? total_orders_value # Track spending ``` --- ## 🔄 Three Core Flows ### Flow A: Direct Customer ``` Browse → Product Page → Place Order → Deliver → Complete (no commission) ``` ### Flow B: Affiliate Referral ``` Affiliate shares link (?ref=code) ↓ Customer clicks link → middleware stores in session ↓ Browse → Product → Place Order ↓ Order created with affiliate attached ↓ Order delivered → Commission calculated! ↓ Affiliate balance updated ``` ### Flow C: Partner Bulk ``` Verified Partner Dashboard ↓ Create Bulk Order (min qty required) ↓ Order created with partner source ↓ Shipped → Delivered ↓ (No affiliate commission) ``` --- ## 💰 Commission Logic **ONLY calculated when:** 1. `order.status == 'delivered'` ✓ 2. `order.source_type == 'affiliate'` ✓ 3. `order.affiliate` is set ✓ **Calculation:** ``` commission = order.total_price * (affiliate.commission_percentage / 100) ``` **Example:** - Order total: $100 - Affiliate commission rate: 10% - Commission earned: $10 --- ## 🛠️ Tech Stack - **Framework:** Django 5.0.3 - **Database:** PostgreSQL (or SQLite for dev) - **ORM:** Django ORM - **Auth:** Django built-in + CustomUser model - **Templates:** Django templates (no JavaScript framework) --- ## 📋 Features Implemented ✅ Custom user model with multiple roles ✅ Product catalog with categories ✅ Order management with source tracking ✅ Affiliate system with unique codes ✅ Commission calculation engine ✅ Affiliate tracking middleware ✅ Partner distribution system ✅ Bulk pricing configuration ✅ Commission payout management ✅ Order status logging ✅ Click analytics tracking ✅ Admin interface for all models ✅ Django management commands ✅ Basic HTML templates ✅ Comprehensive documentation --- ## 🚀 Getting Started ### 1.
